Laravel Installation

  • Install Command : composer create-project laravel/laravel example-app
  • Run Server : php artisan serve

    • Jetstream Installation

  • Install Command : composer require laravel/jetstream
  • Install Jetstream With Livewire : php artisan jetstream:install livewire
  • Install Jetstream With Livewire and Teams : php artisan jetstream:install livewire --teams
    • Finalizing The Installation..... 
                  npm install
              npm run dev
              php artisan migrate
    Liverwire Publish For......
    php artisan vendor:publish --tag=jetstream-views
